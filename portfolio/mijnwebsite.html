Mijn website

In dit hoofdstuk bespreek ik hoe het was om voor het eerst een website te maken.

Hoe heb ik de website opgebouwd?

Voor het programmeren van deze website heb ik gebruik gemaakt van het programma Visual studio Code. In Visual studio Code heb ik voor het programmeren voornamelijk gebruik gemaakt van Bootstrap, HTML en CSS. Als Finance and Control student heb ik nog niet eerder met deze programma’s gewerkt. Voor mij was het dus een leuke uitdaging om deze website in elkaar te zetten.

Bootstrap

Voordat ik aan mijn website begon kreeg ik van Dirk Sanders te horen dat Bootstrap handig was om te gebruiken. Bootstrap is hulpmiddel waar veel voorgeprogrammeerde functies in zitten die makkelijk te gebruiken zijn. Van Bootstrap heb ik een standaard template gedownload wat de basis werd voor mijn website. Vervolgens ging ik bij mijzelf na hoe ik de website op wilde bouwen. Ik wilde een startpagina en vervolgens moet je snel naar mijn opdrachten kunnen. Om snel naar de opdrachten te gaan heb ik een carrousel gemaakt door gebruik te maken van Bootstrap. In eerste instantie lukte het niet om de carrousel juist op mijn webpagina te zetten, uiteindelijk door veel te onderzoeken is het mij gelukt om deze juist op de pagina te laten zien. Dit heeft wel veel tijd gekost, maar het is het wel waard.

HTML

Tijdens deze periode heb ik voor het eerst met HTML gewerkt. Voor Smart Connection heb ik wat workshops gevolgd van Youtube om te begrijpen hoe HTML werkt. Door de workshops te volgen heb ik een goede basis gekregen van kennis over HTML. Door de kennis over HTML kon ik ook makkelijker aan mijn website werken.

In HTML heb ik voornamelijk de opbouw van mijn website gemaakt. Voor iedere webpagina met opdrachten enz. heb ik aparte HTML-bestanden gemaakt, zodat dit overzichtelijk was voor mij.

Met HTML was ik voornamelijk druk mee hoe ik de opdrachten mooi op mijn pagina kreeg. Want er zit verschil tussen opdrachten die je in Word op maakt en hoe je ze op je website weergeeft.

CSS

De CSS wordt gebruikt om je website op te maken. Om de juiste opmaak te krijgen heb ik veel op internet gezocht. Uiteindelijk door veel naar voorbeelden te kijken kreeg ik CSS een beetje door het werkte. De bootstrap opmaak was al goed, ik heb CSS voornamelijk voor kleine opmaak dingetjes gebruikt waar ik niet tevreden over was.

Zelfreflectie

Ik ben trots op mijzelf hoe ik mijn website heb gemaakt. Ik moet toegeven dat ik veel heb van internet en niet zoveel echt zelf heb gecodeerd, maar het zo beter goed gestolen dan slecht verzonnen. Ik vond ook leuk om te maken mijn website, je vindt namelijk steeds nieuwe dingen die beter kunnen. Een website is namelijk nooit echt af. Ik had niet verwacht dat ik zo veel uur met mijn website bezig zou zijn maar het is natuurlijk allemaal wel waard geweest.
