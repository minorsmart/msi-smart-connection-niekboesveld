Persoonlijke doelstelligen

In dit hoofdstuk bespreek ik mijn persoonlijke doelstelling waar ik aan heb gewerkt tijdens de minor Smart Industry.

Mijn persoonlijke doelen voor de minor Smart Industy


In het begin van de minor Smart Industry heb ik drie doelstellingen opgesteld die ik tijdens deze minor wil behalen.

# Doel Activiteit Normen Tijd
1 Technologische kennis op doen Workshops tijdens de minor volgen. Hier vervolgens de juiste kennis meenemen die ik kan gebruiken in de toekomst. Ik vind dat de doelstelling behaald is als ik terug kan kijken op de minor met het gevoel dat ik de juiste kennis heb opgedaan. Deze doelstelling heb ik voor deze minor opgesteld.
2 Op persoonlijk vlak ontwikkelen Tijdens deze minor wil ik aan mijn zelfvertrouwen werken. Ik ben normaal gesproken niet de persoon die als eerst iets zegt. Hier heb ik geen norm voor omdat ik al een langere tijd mijzelf probeer te verbeteren. Aan het einde van mijn opleiding wil ik een professional zijn die een bedrijf wil hebben.
3 Individuele opdrachten goed uitwerken. Ik vind het lastig op individuele opdracht goed uit te werken. Tijdens de minor wil ik opdrachten goed individueel uitwerken. Dit komt omdat er veel individuele opdrachten zijn. Mijn portfolio moet er professioneel uit komen te zien. Ik beoordeel zelf of ik de norm haal. Deze doelstelling heb ik voor deze minor opgesteld.

Eindreflectie

In deze eindreflectie ga ik terugkijken op de doelstellingen die ik in het begin van de minor heb opgesteld.

Voor mij persoonlijk heb ik veel technische kennis opgedaan tijdens de minor. Ik ben veel te weten gekomen over hoe je een website moet maken. Een website maken vond ik erg leuk om te doen, hier heb ik veel plezier mee gehad. Daarnaast heb ik veel geleerd over hoe een IOT-systeem werkt die gebruik maakt van sensoren. In het begin vond ik het lastig om het te begrijpen, maar hoe verder ik kwam met het programmeren hoe beter ik snapte wat iedere sensor precies doet. Hier heb ik ook veel plezier aan beleeft.

Tijdens de periode die ik bij Flynth aan het dashboard heb ik mogen werken heb ik ook veel mogen leren over technologie. Ik heb mee kunnen kijken wat er allemaal omtrent de data moet gebeuren voordat het in een dashboard kan komen. Deze doelstelling is dan ook zeker voor mij persoonlijk gehaald.

Mijn persoonlijke ontwikkeling kon ik niet helemaal zoals ik mij had voorgesteld halen. Door covid-19 moesten wij veel thuis werken, hierdoor kon ik niet veel in een groep fysiek mijn aanwezigheid tonen. Toch vind ik dat ik wel ben gegroeid tijdens de minor, ik moest nieuwe connecties leggen tijdens den minor omdat ik nog niemand kende.

Ik vind dat ik op mijn persoonlijke ontwikkeling nog steeds kan verbeteren, maar ik ben deze minor al zeker wat verbeterd.

In het begin van de minor vond ik het lastig om individueel een literatuuronderzoek te maken. Dit had ik tijdens mijn opleiding nog niet eerder gedaan. Tijdens de minor ben ik hierin verbeterd, ik vind dat je dit ook terug kan zien in mijn onderzoeken bij Smart Start. Elk onderzoek werd door 5 medestudent beoordeeld. Ik kreeg fijne feedback die ik steeds kon toepassen in het volgende literatuuronderzoek. Ik kan zeggen dat ik persoonlijk vind dat ik deze doelstelling heb behaald. Ik zie dat ik mij heb verbeterd en dat de kwaliteit van mijn onderzoeken ook steeds omhoog is gegaan.
