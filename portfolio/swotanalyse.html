Wat is een SWOT-analyse?

Een SWOT analyse is een populaire 4-box strategie analyse tool.

De SWOT analyse bestaat uit de volgende 4 kenmerken:

  • Strenghts(Sterktes)
  • Weaknesses(Zwaktes)
  • Opportunities(Kansen)
  • Threats(Bedreigingen)
De SWOT-analyse wordt al jaren gebruikt en is een van de meest gebruikte strategische tool van de moderne tijd. Deze tool wordt gebruikt door allerlei soorten ondernemingen zoals, industriële ondernemingen, commerciële ondernemingen en goede doelen. (Sarsby, 2016)

De SWOT kan niet gebruikt worden om een strategie op te stellen. De SWOT is een analyse en ontwikkel tool. Je moet namelijk nog steeds beslissingen maken welke strategie je gaat hanteren. (Sarsby, 2016)

Zwaktes

Mijn grootste zwakte is dat ik het lastig vind om te communiceren met personen die ik nog niet goed ken. Hier heb ik eerder wel last van gehad tijden mijn stages. Het kwam vaak voor dat ik stil was en niet direct op iemand af durfde te stappen. Ik was vaak bang voor de reactie van de persoon.

Ook vind ik het lastig om onderzoeken zelfstandig op te stellen. Tijdens mijn opleiding hebben we veel in projectgroepen gewerkt en ik heb weinig zelfstandige onderzoeken gedaan. Hierin kan ik mij zeker ontwikkelen.

Soms vind ik het lastig om mijn concentratie bij de opdracht te houden als ik thuis werk. Ik raak dan vaak afgeleid als ik een opdracht maak. Ik vind het altijd fijn als ik op school kan werken, want ik kan dan minder snel afgeleid raken.

Kansen

Ik zie vooral kansen tijdens deze minor om mijzelf te ontwikkelen op verschillende gebieden. Ik zie er vooral naar uit om mij te ontwikkelen op technisch gebied. Ik wil de stof zoveel mogelijk tot mij nemen zodat ik dit later kan toepassen in mijn vakgebied. Daarnaast wil ik mijzelf ontwikkelen op mijn communicatie. Ik wil tijdens het project goed communiceren met de opdrachtgever en gewoon mijzelf zijn.

Bedreigingen

Op dit moment heb ik weinig ervaring op technisch gebied. Een valkuil kan zijn dat de stof te lastig is en het niet lukt om dit tot mij te nemen.
